✅ SOAP 1.2 Employee Web Service

The service is deployed and running on GlassFish 7 Full Platform with Jakarta EE 10.

WSDL:
http://localhost:8080/EmployeeService/EmployeeService?wsdl
SOAP 1.2 Endpoint:
http://localhost:8080/EmployeeService/EmployeeService

Available Operations

  • getEmployee(employeeId) — Retrieve one employee
  • getAllEmployees() — List all employees
  • createEmployee(employee) — Create a new employee
  • updateEmployee(employee) — Update an existing employee
  • deleteEmployee(employeeId) — Delete an employee

Test with cURL (SOAP 1.2)

curl -X POST \
  http://localhost:8080/soap-service/EmployeeService \
  -H "Content-Type: application/soap+xml;charset=UTF-8" \
  -d '<soap:Envelope xmlns:soap="http://www.w3.org/2003/05/soap-envelope"
                    xmlns:ser="http://service.soap.enterprise.com/">
    <soap:Body>
      <ser:getAllEmployees/>
    </soap:Body>
  </soap:Envelope>'
